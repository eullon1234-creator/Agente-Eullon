# Agentes Eullon

Repositorio simples para guardar e organizar todos os agentes.

## Estrutura

- agents/ - cada agente em sua propria pasta
- docs/ - documentacao geral
- templates/ - modelos reutilizaveis

## Como usar

1. Crie uma pasta para cada agente em agents/.
2. Use o modelo em templates/agent.md para documentar o agente.

## Baixar com Git

1. Abra o terminal no VS Code.
2. Rode:

```bash
git clone <URL_DO_REPOSITORIO>
```

3. Entre na pasta:

```bash
cd Agente-Eullon
```

## Usar no VS Code

1. No VS Code, clique em File > Open Folder e escolha a pasta do repositorio.
2. Abra o arquivo do agente em agents/ e copie o prompt.
3. Cole o prompt no chat do Copilot ou no local onde voce cria seus agentes.
